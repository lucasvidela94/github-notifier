# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-XX-XX

### Added
- Initial release
- System tray integration with bell icon
- GitHub Notifications API polling
- Desktop notifications via notify-send
- SQLite local persistence
- Systemd user service support
- Click-to-open comment links
- Automatic mark as resolved

### Features
- Poll interval configuration
- Notification filtering (comments, mentions, review requests)
- Tray icon states (idle vs. has notifications)
- Up to 15 tray menu entries
- Activity type prefixes in menu
